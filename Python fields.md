- Data Analyst (The Insight Reporter)[interpret data to create actionable insights, dashboards, and reports]
    - Focus: Data visualization, reporting, exploratory data analysis, and dashboarding.
    - Key Skills: SQL, Excel, Tableau, Power BI, Python/R (pandas).
    - Goal: Interpret past data to guide immediate business decisions. 

- Data Engineer (The Infrastructure Builder)[build the infrastructure and data pipelines]
    - Focus: Data pipeline construction, ETL (Extract, Transform, Load) processes, data warehousing, and scalability.
    - Key Skills: SQL, Python, Scala, Spark, Kafka, Apache Airflow, Snowflake, Cloud (AWS, GCP, Azure).
    - Goal: Provide reliable, clean, and organized data for the team. 

- Data Scientist (The Modeling Expert)(Create Predictive Models and ML Algorithms)
    - Focus: Predictive modeling, machine learning, statistical analysis, and algorithm development.
    - Key Skills: Python/R, Machine Learning algorithms, Statistics, SQL, TensorFlow, PyTorch, Pandas.
    - Goal: Predict future outcomes and discover hidden, complex patterns. 

- Key Differences Summary
    - Complexity/Skill/Salary: Data Analyst (Lowest coding) < Data Engineer < Data Scientist (Highest math/ML).
    - Data Flow: Engineers set up the pipeline, Scientists use the data to model, Analysts use the data to report.
